# Node.js Practice 🚀

This repository contains my hands-on practice with **Node.js** to build a strong foundation in backend development.

## 📌 Description
This project demonstrates core Node.js concepts like creating servers, handling HTTP requests, routing, and building REST APIs without using external frameworks.

## ⚙️ Features
- Basic HTTP server creation  
- REST API methods (GET, POST, PUT, DELETE)  
- CRUD operations  
- JSON data handling  
- Asynchronous programming (callbacks, promises, async/await)  

## 🛠️ Technologies Used
- Node.js  
- JavaScript
